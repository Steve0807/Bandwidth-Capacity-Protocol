## 更多内容

Wiki将使用文章中的大部分定义，但同时对文章中的部分定义做出了微调：
* 将公网环境也称之为Easy NAT
* 仅将Symmetric NAT称为Hard NAT，不考虑其他特殊NAT类型情况。
